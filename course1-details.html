HTML

HTML Class Overview


This is the introductory course from our offering of various HTML classes and CSS training courses. If you are new to web design or web development, this is the best place to start. This HTML training course combines practical hands-on exercises with an overview of the HTML and CSS code and tools necessary for building and maintaining web sites. Best practices including understanding how to develop for multiple browsers and devices is emphasized.
